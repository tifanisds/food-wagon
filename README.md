# 🍔 Food Wagon

Bem-vindo ao **Food Wagon**! Este é um projeto desenvolvido com o objetivo de praticar **HTML**, **CSS**, **Flexbox** e **Grid Layout**.

## 🚀 Tecnologias Utilizadas
- **HTML5** → Estruturação do conteúdo
- **CSS3** → Estilização do site
- **Flexbox** → Layout flexível
- **Grid Layout** → Organização da página

## 🎯 Objetivo
Este projeto é **puramente educacional**, focado no aprimoramento das habilidades em **layout responsivo** e boas práticas no desenvolvimento front-end.
